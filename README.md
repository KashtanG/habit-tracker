# 🎯 Трекер привычек (Habit Tracker)

Консольное приложение для отслеживания ежедневных привычек. Проект создан для закрепления базовых навыков Python.

## 🚀 Функционал
- ➕ Добавление новых привычек
- ✅ Отметка выполнения привычки за сегодня
- 📊 Просмотр статистики (процент выполнения за неделю)
- 🔥 Серия выполнений привычки
- 📈 Визуализация прогресса в консоли
